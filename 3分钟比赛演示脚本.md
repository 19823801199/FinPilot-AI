# 3分钟比赛演示脚本

> FinPilot AI — AI专家团队驱动的金融智能工作台
> 演示时长：3分钟（180秒）| 镜头数：18 | 每10秒一个镜头

---

## 演示节奏设计

| 时间段 | 目标 | 核心展示 |
|--------|------|---------|
| 0:00-0:30 | 评委理解项目价值 | 工作台首页 + 一句话定位 |
| 0:30-1:30 | 评委看到AI能力 | 股票分析 + Multi-Agent + RAG |
| 1:30-2:30 | 评委看到完整闭环 | 学习助手 + 知识库问答 + 跨功能联动 |
| 2:30-3:00 | 评委记住产品 | 收尾总结 + 品牌印象 |
