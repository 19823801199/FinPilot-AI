# FinPilot AI - 更新日志

## [1.0.0] - 2026-06-23 - 正式发布版

### 新增内容

#### AI 智能工作台
- 集成 AI 对话工作台，支持多轮对话与上下文记忆
- 实现流式输出（SSE），实时显示 AI 回复内容
- 会话管理系统：新建、历史记录、删除、搜索
- 消息 Markdown 渲染，支持代码块、表格、数学公式
- 会话分组与标签筛选

#### 股票研究员
- 股票搜索模块：支持 A 股代码与名称模糊搜索
- 股票详情展示：实时行情、K 线图（Recharts）、基本面数据
- AI 智能分析：基于 DeepSeek API 生成深度研究报告
- 自选股管理：关注/取消关注，价格变动提醒
- 历史分析记录保存与对比

#### 知识库与 RAG 检索增强生成
- 文档上传与管理：支持 PDF、Markdown 格式
- ChromaDB 向量数据库集成，实现语义检索
- RAG 检索增强生成：基于知识库内容进行智能问答
- 文档智能分割与索引管理
- 知识库内容全文检索

#### 多 Agent 协作系统
- 学习专家 Agent：个性化学习路径规划
- 研究专家 Agent：股票深度分析与报告生成
- 知识专家 Agent：文档理解与知识问答
- Agent 任务编排与协同调度
- Agent 对话记录与结果汇总

#### 金融学习中心
- 系统化金融课程体系，覆盖 431 金融综合考点
- 章节知识点讲解与配套练习
- 智能练习系统：选择题、判断题、简答题
- 错题本与学习进度追踪
- 学习统计面板：正确率、学习时长、掌握度分析

#### 流式聊天与交互体验
- 全双工流式聊天，打字机效果实时展示
- 对话中断、重新生成、继续生成等交互控制
- 聊天历史即时保存与恢复
- 拖拽上传文件到聊天区

#### Docker 全面支持
- 前端多阶段 Docker 构建（web.Dockerfile）
- 后端多阶段 Docker 构建（api.Dockerfile）
- Docker Compose 一键编排前后端服务
- 环境变量注入与容器化配置

### 修改内容

- **版本升级**: 从 v0.1.0-RC2 正式升级至 v1.0.0
- **依赖清理**: 移除了项目中未使用的 npm 和 pip 依赖，减小构建体积约 30%
- **目录结构调整**: 优化 Monorepo 目录结构，统一命名规范，移除冗余文件夹
- **组件重构**: 将多处重复的 UI 逻辑提取为通用组件，提升代码复用率
- **性能优化**: 优化 Zustand store 的订阅机制，减少不必要渲染

### 修复内容

- **安全漏洞**: 修复 chat-area 组件中存在的 XSS 跨站脚本漏洞，增加输入消毒与转义处理
- **类型定义**: 消除多处重复的 TypeScript 类型定义，统一导入入口
- **孤立目录**: 清理项目中不再引用的孤立目录和废弃文件
- **模拟数据**: 移除前端中残留的 Mock 数据占位，确保正式 API 对接完整
- **API 路由**: 修复部分 API 路由未正确注册的问题
- **状态管理**: 修复 Zustand store 中部分状态未正确清理导致的内存泄漏

### 设计决策记录

1. **Next.js 15 + React 18**: 选择稳定 LTS 版本以保证生产环境可靠性
2. **TailwindCSS v4 + Shadcn UI**: 使用 CSS 原生变量定义设计系统，保持样式一致性
3. **SQLite 开发 / PostgreSQL 生产**: MVP 阶段使用 SQLite 降低环境依赖
4. **Zustand 状态管理**: 轻量、高性能，按功能模块拆分 store
5. **ChromaDB 向量数据库**: 开源、轻量级，适合知识库 RAG 场景
6. **DeepSeek API**: 高性能推理，支持 Function Calling，适合多 Agent 场景
